My Work

Honeypot: Project to Demonstrate Network Security Systems

The purpose of this project was to identify one of the eight CISSP domains that I had a keen interest in. Within that domain, I narrowed down my topic and chose to do the project on the honeypot security concept to explore in depth. This project consists of a step-by-step on how to set up a honeypot and expose it to the internet to try and collect data traffic and information on adversaries. It is also recommended that the use of honeypot should be used as enticement and not entrapment. The pictures shown are some commands I used for the installation process. Additionally, this project was a highlight, and was used as an example for other courses.

Smart Doorbell: Device Penetration Testing

The purpose of this project was to conduct research and further analysis on the selected device. We have found from part one of the reports that the processor on the device that manages the input/output as well as the system itself, runs on FreeRTOS. Further research suggests that the device is using WPA/WPA2 and WEP, which we considered WEP unsecure for a doorbell. In addition to learning about the components, and analyzing them, we captured data on Wireshark to see the communication that occurred with other devices, such as the laptop that was hosting the WiFi hotspot with the doorbell and the application on the phone. Pentesting the device connected to WPA2 was not penetratable, but WEP was. We were commended for our success in the pentesting project and the additional work and information that we found.

It helps and guides others with similar alerts make sound decisions to escalate the incident or mark it as FP. Client Monthly Report

I was tasked to do the client's monthly report. The report consists of sorting the critical alerts first, exporting the file to excel, and lastly adding the table to its appropriate page on Stellar Cyber. The monthly report shows the overall alerts within the month ranging from critical to non-critical. It provides the client a more in depth analysis, and provides us the opportunity to sit with the client and discuss tuning of alerts and actions to be done.
